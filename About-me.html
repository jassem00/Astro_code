What is Astro :

Astro is the first programming language of its kind that simulates algorithms that are taught in Tunisian educational institutions, and although in its first version it represents only 20% of algorithms, but after much development, it will become a helpful environment for both students and teachers to find solutions to algorithms in the shortest time and with the highest possible quality.

Start coding :

Step 3 : create a folder and paste the files in

Step 4 :  Open "Astro.txt" and write your code there

What is the purpose of this project ?

I decided to take a step back and think about the main issues that will dominate the teaching of programming and algorithms in the coming years as technology develops, and how a new language can help address it and link both algorithms and programming languages
